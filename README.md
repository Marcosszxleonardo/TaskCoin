# 📝 TaskCoin

O TaskCoin é um aplicação web que usa recompensas para incentivar crianças e adolescentes a cumprir tarefas, ajudando a desenvolver disciplina e responsabilidade.

# 🧠 Sobre o Projeto

O TaskCoin é uma Aplicação web desenvolvido com foco em aprendizado e prática de tecnologias modernas como React e pensamos futuramente implementar React Native. O TaskCoin é voltado para a criação de uma plataforma interativa, responsiva e intuitiva.
O sistema permite que pais e responsáveis organizem tarefas para seus filhos, enquanto crianças e adolescentes podem visualizar, realizar e acompanhar suas atividades. Entre as principais funcionalidades estão:

⦁ Cadastro de usuários (pais e filhos)
⦁ Criação e gerenciamento de tarefas
⦁ Marcação e validação de tarefas concluídas
⦁ Sistema de pontos como recompensa
⦁ Troca de pontos por recompensas personalizadas
⦁ Acompanhamento do progresso

Com um layout responsivo e componentes reutilizáveis, o TaskCoin transforma uma simples lista de tarefas em uma experiência gamificada, incentivando disciplina, organização e responsabilidade de forma prática e envolvente.
