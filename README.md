# 🧠 Sobre o Projeto

O **TaskCoin** é uma aplicação web desenvolvida com foco em aprendizado e utilização de tecnologias modernas para desenvolvimento web.

O sistema foi criado para ajudar pais e responsáveis a incentivarem hábitos positivos em crianças e adolescentes por meio de uma experiência gamificada baseada em tarefas e recompensas.

A plataforma permite que responsáveis criem tarefas, acompanhem o progresso dos filhos e validem atividades concluídas. Já os filhos podem visualizar tarefas, acumular pontos e trocar recompensas personalizadas.

O objetivo do projeto é transformar atividades do cotidiano em desafios motivadores e educativos.

---

# 🎯 Objetivo do Sistema

O TaskCoin busca auxiliar famílias na organização de rotinas e no incentivo à responsabilidade, utilizando gamificação como ferramenta de motivação.

Além disso, o projeto também possui finalidade acadêmica, proporcionando experiência prática em desenvolvimento frontend, backend e integração de sistemas.

---

# ✨ Funcionalidades

- Cadastro de usuários (pais e filhos)
- Sistema de login
- Criação e gerenciamento de tarefas
- Marcação de tarefas concluídas
- Validação de tarefas pelos responsáveis
- Sistema de pontos (TaskCoins)
- Sistema de recompensas
- Troca de pontos por benefícios
- Acompanhamento de progresso
- Interface responsiva e intuitiva

# 🏗️ Arquitetura do Sistema

O sistema foi dividido em três camadas principais:

- **Frontend:** responsável pela interface do usuário
- **Backend:** responsável pelas regras de negócio e API
- **Banco de Dados:** responsável pelo armazenamento das informações

A comunicação entre frontend e backend ocorre através de uma API REST.

# 🎮 Funcionamento do Sistema

1. O responsável cria uma tarefa  
2. O filho visualiza a atividade  
3. A tarefa é realizada  
4. O filho marca como concluída  
5. O responsável valida a tarefa  
6. O sistema adiciona TaskCoins  
7. Os pontos podem ser trocados por recompensas

# ▶️ Como Executar o Projeto

## 1. Clonar o Repositório

```bash
git clone https://github.com/Marcosszxleonardo/TaskCoin
```

## 2. Acessar a Pasta do Projeto

```bash
cd TaskCoin
```

## 3. Instalar as Dependências

```bash
npm install
```

## 4. Executar o Frontend

```bash
npm run dev
```

## 5. Executar o Backend

```bash
mvn spring-boot:run
```
